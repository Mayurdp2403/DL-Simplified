# Dataset Details

## Overview

This project uses the **arXiv Scientific Research Papers Dataset** for building a Research Paper Recommendation System using NLP and Machine Learning techniques.

The dataset contains scientific research papers collected from arXiv, including titles, abstracts, categories, authors, and metadata. It is useful for tasks such as recommendation systems, text classification, semantic search, topic modeling, and research trend analysis. :contentReference[oaicite:0]{index=0}

---

## Dataset Source

Dataset Link:  
https://www.kaggle.com/datasets/sumitm004/arxiv-scientific-research-papers-dataset

---

## Dataset Files

### `Train.csv`
Contains the training data used for model building and recommendation learning.

### `test.csv`
Contains testing data used for evaluating model performance.

### `OutPutFile_Submission.csv`
Stores the final prediction/recommendation outputs generated by the model.

---

## Features Included

The dataset includes information such as:

- Research Paper Title
- Abstract / Summary
- Authors
- Categories
- Publication Information
- Keywords and Metadata

---

## Project Usage

This dataset is used to:

- Recommend similar research papers
- Perform semantic similarity search
- Generate embeddings using BERT
- Apply TF-IDF vectorization
- Perform FAISS-based retrieval
- Build NLP-based recommendation pipelines
- Perform classification using Random Forest
